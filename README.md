# Smart Tuition Center Website

Professional Tuition Center Website - Fiverr Ready.

## Features
- Home, Courses, About, Contact Pages
- Responsive Design (Mobile & Desktop Friendly)
- Interactive Buttons with Hover Effects
- WhatsApp Quick Contact Button

## Files Included
- index.html
- style.css
- Images folder (hero.jpeg, teacher.jpeg, etc.)

## Description
This is a professional tuition center website designed for small businesses and online tutors. 
The website is fully responsive and includes interactive elements like buttons with hover effects 
and a WhatsApp quick contact button for instant communication with clients.
